- Multiplayer game

## Run Locally

Clone the project

```bash
  git clone https://github.com/jcdesignweb/chess
```

Go to the project directory

```bash
  cd chess
```

Install dependencies

```bash
  npm install
```

Run vite

```bash
  npm run dev
```
